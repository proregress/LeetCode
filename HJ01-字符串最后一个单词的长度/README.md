### 1.next()和nextLine()
#### nextLine()方法：一次读取一行数据，只在回车符时终止录入；
#### next()方法：一次读取一个无间隔子串，比如TAB、空格、回车符的时候，会终止扫描；

### 2. length和length()
  String类可以定义字符串*变量*和字符串*数组*  
#### length - 字符串数组的长度
#### length() - 字符串对象的长度
