### 1. Java的基本数据类型包括八种:   
- 整数类型：byte，short，int，long
- 浮点数类型：float，double
- 字符类型：char
- 布尔类型：boolean

  #### 类型从小到大依次为：

  ​ byte short int long float double

  ​ (char)

### 2. JAVA基本类型之间转换的两种方式：   
#### (1) 自动（隐式）类型转换：从小类型到大类型，不需要强制转换符
#### (2) 强制类型转换：从大类型到小类型，需要强制转换符实现强制转换   
  强制转换符：（需要转换成的类型）变量
